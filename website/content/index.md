# UOR Foundation

The Universal Object Reference (UOR) Framework is a formal mathematical framework
for content-addressed, algebraically-structured object spaces.

**Key features:**
- Ring substrate Z/(2^n)Z with canonical two-involution structure
- Dihedral group D_{2^n} generated by ring negation and bitwise complement
